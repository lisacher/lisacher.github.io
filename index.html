輕鬆部署 Github Page

當網頁工程師最期待的就是把網頁部署到線上！
只要前端的網頁是靜態檔案，不需要搭配固定的伺服器，就可以輕鬆把網頁部署在雲端上！

打包 Vue 檔案

在終端機輸入：

$ npm run build

Vue CLI 就會自動幫我們把所有寫好的 Vue 組件和設定等等一併作打包的動作，打包好了之後，在專案資料夾中，你會看到多了一個 dist 資料夾，裡面放的就是靜態網頁所需的所有檔案
